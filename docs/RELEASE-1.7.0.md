## 🧁 CupCat 1.7.0

The free, local, AI-native video editor for Windows. Claude edits your timeline; almost everything
runs offline on your PC. No credits, no watermark, no account.

### Install

1. Download **`CupCat_1.7.0_x64-setup.exe`** below and run it.
2. It's self-contained — every local engine (ffmpeg, Whisper, Piper, sherpa-onnx) is bundled. No preinstalls, no Python.
3. To use the AI assistant (in-app chat or MCP), sign in to **Claude Code** on the same machine.
4. To generate media, sign in to **Higgsfield** (`higgsfield auth login`).

### What's new in 1.7.0

**A redesigned, polished editor UI** — folder tiles with drill-down and drag-to-organize, an
auto-growing and resizable assistant panel, a timecode ruler with film-strip clips and waveforms,
source tabs above the monitor.

**An expanded local AI toolbox — free and offline:**

- **Rough cut** — turn a folder of footage into an assembled first cut with one command.
- **Transcript editing** — edit the video by editing its words; one-click removal of filler ("um / uh / cioè …").
- **Auto-reframe** — reframe to 9:16 / 1:1 by tracking the subject, shot by shot.
- **Stem separation** — split any clip's audio into voice and music.
- **Smooth slow-motion** — fluid, motion-interpolated slow-mo.
- **Motion tracking** — pin text and stickers to a moving subject.
- **Dubbing** — transcribe → translate → speak → time-fit a dub track in another language.
- **Generative transitions & motion graphics** — describe it and the assistant animates it.
- **Command palette** (Ctrl+K) — run any action, or hand the request to the assistant.
- **AV1 export** (10-bit), alongside H.264 / H.265 / HDR HEVC / ProRes / FCPXML / lossless.
- **Headless CLI** — batch-render projects from the command line.

Plus a project-template system, semantic media search, and a headless render CLI. Around 100 editing
tools are now available to the AI assistant, in-app or over MCP.

### Requirements

- Windows 10 / 11 (x64)
- Claude Code signed in (for the AI assistant / MCP)
- A Higgsfield account (only if you use generative media)
